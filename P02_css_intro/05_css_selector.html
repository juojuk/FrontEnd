Visos modernios svetainės ir web sistemos naudoja šias tris pagrindines technologijas: HTML, CSS ir JavaScript. Tai yra web kalbos.

Kas yra HTML?

HTML verciasi HyperText Markup Language. yra pats pagrindinis žiniatinklio kūrimo elementas. Jis apibrėžia interneto turinio prasmę ir struktūrą. Kitos technologijos, išskyrus HTML, paprastai naudojamos aprašyti tinklalapio išvaizdą / pristatymą.
